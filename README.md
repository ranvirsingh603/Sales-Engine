# Sales Engine: Automated CRM, Workflow Automation & Looker Studio BI System

## 🎯 Project Overview & Business Value
In high-velocity B2B sales environments, relying on physical diaries or manual spreadsheets inevitably leads to missed follow-ups, opaque pipelines, and lost revenue. This project showcases the end-to-end design, development, and implementation of **Sales Engine**—a centralized, fully automated CRM and Management Information System (MIS). 

By transforming a legacy, manual sales tracking workflow into an event-driven automated ecosystem, this solution establishes absolute pipeline visibility from lead ingestion to final order conversion.

* **Key Deliverable:** Integrated lead-tracking engine and real-time operational Looker Studio dashboards.
* **Target Impact:** Elimination of manual sales tracking delays, automated document generation, and real-time operational bottleneck visibility for executive leadership.

---

## 🏗️ System Architecture & Event-Driven Automation
The system replaces disjointed operations with a unified, automated data architecture designed to enforce tracking velocity:

`

### 1. The Full-Stack Infrastructure

* **Data Ingestion Engine:** Automated front-end Google Forms deployment to standardize field entry (Inquiry details, client criteria, assignment parameters) directly from team touchpoints.
* **Relational Database Core:** Google Sheets configured as a centralized ledger handling system states, relational remarks history, and active/inactive status parameters.
* **Event-Driven Automation Core:** Custom JavaScript-based **Google Apps Script** architecture handling backend system logic, automated data synchronization, time-based triggers, and conditional data routing.
* **Automated Document Generation Engine:** Implemented programmatic text interpolation using Apps Script to dynamically compile cell data, generate formal client Microsoft Word/Google Doc proposals on-the-fly, and automatically log the generated document URLs back to the master database.

## 📊 Business Intelligence & Looker Studio Dashboards

The visualization layer processes raw operational log streams into three distinct, high-impact monitoring dashboards to maintain absolute internal accountability and zero-delay pipelines:

### 1. Daily Due Follow-up Dashboard

* **Operational Focus:** Filters out pipeline noise to present sales executives with *only* the specific actionable leads requiring touchpoints on the current calendar day.
* **Business Value:** Prevents lead leakage and guarantees no strategic customer engagement falls through the cracks due to administrative oversight.

### 2. Delayed Tasks & Accountability Dashboard

* **Operational Focus:** Automatically flags broken SLA metrics, highlighting missed milestones and overdue actions across individual sales representatives.
* **Business Value:** Provides management with instant, objective visibility into operational bottlenecks and personnel friction points.

### 3. Proposal-to-Order Conversion Engine

* **Operational Focus:** Tracks metrics post-proposal delivery, detailing historical interactions, deal value thresholds, and pending client remarks.
* **Business Value:** Maximizes pipeline win rates by focusing late-stage closing efforts on highly qualified deals lingering in late-stage negotiations.

* **Accelerated Decision Cycles:** Eliminated executive blind spots by replacing manual inquiry reviews with automated, real-time funnel metrics.
